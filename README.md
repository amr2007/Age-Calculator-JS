# Age Calculator (JavaScript)

A simple and accurate age calculator built using HTML, CSS, and JavaScript.  
It calculates the user's age based on their date of birth and the current date provided by the Luxon time library.

## Features

- User-friendly interface
- Accurate age calculation using the current date
- Automatically adjusts if the birthday has not occurred yet this year
- Clean and minimal UI design

## Technologies Used

- HTML
- CSS
- JavaScript
- Luxon (for date and time handling)

## How It Works

1. The user selects their date of birth.
2. The application retrieves the current date using Luxon.
3. The age is calculated as the difference between the current year and birth year.
4. If the birthday has not yet occurred this year, one year is subtracted.
5. The final age is displayed to the user.
